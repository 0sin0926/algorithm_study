### 1. 문제 이해
- n이 주어졌을 때, n보다 크고 2n보다 작거나 같은 소수 개수 구하기
  
### 2. 입출력 조건
- 1초/256MB
- bool 배열과 int 입력
- n <= 123,456

### 3. 문제 풀이
- 에라토스테네스의 체로 소수 구하기 -> O(NlglgN)이므로, 그냥 처음부터 250,000개의 숫자의 소수 여부를 bool배열에 저장해놓고 시작해도 무방

### 4. 구현
- 전역변수 bool check[250,000];
- main문에서 에라토스 구현하고
- 그냥 탐색하는 함수 바깥에다가 구현
